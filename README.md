# HR Analytics Dashboard 📊

## Overview
Interactive Power BI dashboard analyzing employee attrition 
and HR metrics for 1,470 employees.

## Tools Used
- Power BI (3-page interactive dashboard)

## Key Findings
- Overall attrition rate: 16%
- Sales department has highest monthly income
- Employees aged 25-35 have highest attrition
- Single employees tend to leave more than married ones

## Dataset
IBM HR Analytics Employee Attrition Dataset (Kaggle) - 1,470 employees
